ALL ROUTE API

GET (Récupération des données)

  • Tous les sujets : http://localhost:6767/topics
  • Tous les commentaires : http://localhost:6767/comments
  • Tous les tags : http://localhost:6767/tags

POST (Création et actions)

  • Login : http://localhost:6767/users/login
  • Créer / S'inscrire (User) : http://localhost:6767/users
  • Créer un sujet (Topic) : http://localhost:6767/topics
  • Voter un sujet (Like/Dislike) : http://localhost:6767/topics/like
  • Créer un commentaire : http://localhost:6767/comments
  • Voter un commentaire (Like/Dislike) : http://localhost:6767/comments/like
  • Créer un tag : http://localhost:6767/tags

PUT (Modification)

  • Bannir un utilisateur : http://localhost:6767/users/ban
  • Modifier un sujet : http://localhost:6767/topics

DELETE (Suppression)

  • Supprimer un utilisateur : http://localhost:6767/users
  • Supprimer un sujet : http://localhost:6767/topics
  • Supprimer un commentaire : http://localhost:6767/comments
